Essential Commands

File Management
# Create directory mkdir directory_name # Copy files cp source destination # Move/rename files mv source destination # Remove files/directories rm file_name rm -r directory_name

Docker

Container management and deployment

Docker Docs GitHub Guide
Basic Commands
# Install Docker sudo pacman -S docker # Start Docker service sudo systemctl start docker sudo systemctl enable docker # Add user to docker group sudo usermod -aG docker $USER
Container Management
# Run a container docker run -d -p 80:80 nginx # List containers docker ps docker ps -a # Stop container docker stop container_id # Remove container docker rm container_id

Kubernetes

Container orchestration and cluster management

Kubernetes Docs GitHub Guide
Installation & Setup
# Install kubectl sudo pacman -S kubectl # Install Minikube curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64 sudo install minikube-linux-amd64 /usr/local/bin/minikube # Start Minikube minikube start # Verify installation kubectl cluster-info
Pod Management
# Create pod kubectl create -f pod.yaml # List pods kubectl get pods # Pod details kubectl describe pod pod-name # Delete pod kubectl delete pod pod-name
